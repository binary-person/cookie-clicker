Make your own translation

  1. Download translation file
  2. Edit using 'utf-8'
  3. Put anywhere on The Internet
  4. Input URL
  5. Play
  6. if you add URL on this spreadsheet, I will made it selectable.

Example

Input URL: https://googledrive.com/host/0ByAtt-u9ddI8d3RSbEtLWllsUWc/trans_ja.js
Play URL: https://googledrive.com/host/0ByAtt-u9ddI8d3RSbEtLWllsUWc/?langfile=https%3A%2F%2Fgoogledrive.com%2Fhost%2F0ByAtt-u9ddI8d3RSbEtLWllsUWc%2Ftrans_ja.js
